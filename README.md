# 🐟 Ryby našich řek a rybníků

Interaktivní webová aplikace o českých sladkovodních rybách. Projekt slouží jako výuková pomůcka do přírodovědy pro děti na základní škole.

Web vznikl jako školní projekt žáka 4. třídy. Pomáhá dětem poznávat ryby hravou formou pomocí kartiček, kvízů a jednoduchých her.

## Co aplikace umí

- 🐟 **Kartičky ryb** – informace o rybách, kde žijí, co jedí a čím jsou zajímavé.
- 🔍 **Poznávačka** – dítě poznává ryby podle obrázků.
- ❓ **Kvíz** – otázky o sladkovodních rybách.
- 🔧 **Postav rybu** – hra o částech rybího těla a jejich funkci.
- 🎮 **Plav a přežij** – mini hra, ve které rybka sbírá potravu a vyhýbá se nebezpečí.
- 🏆 **Online žebříček** – ukládání výsledků pomocí přezdívky.
